Setting it up¶

  1. Create a directory called vintage in your template directory
  2. Create a new template called default.html in that directory

Creating a page manually¶

  1. Click on the new page button in the admin.
  2. Enter the URL path of the old page. Should be everything after the http://domainname.com. For Example: /articles/2010/may/01/bigfoot-sighted/.
  3. Enter the page’s title.
  4. View the source of the page you are archiving.
  5. Select the appropriate HTML content (such as everything between the <body> tags).
  6. Paste it into the content field.
  7. Select a template (leave blank for default) see How Django Vintage selects the template
  8. Enter in any of the page’s metadata fields: page_id, image, description, keywords, and author.

Importing a page¶

Need: URL and CSS selector (like body or #main-content) Does: Loads the page, imports the content based on the CSS selector, sets the title, URL and metadata fields based on the page attributes.

How the metadata fields work¶

The metadata fields, page_id, image, description, keywords, and author, get included in the template context for rendering in any <meta> tags.
